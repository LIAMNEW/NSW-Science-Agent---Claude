# Google Cloud Vertex AI Test Code Generation Prompts Template

This template lets you easily Test Vertex AI Code Generation prompts

For more details you can visit the [API Documentation page](https://cloud.google.com/vertex-ai/docs/generative-ai/code/test-code-generation-prompts)

## Prerequisites and getting started 
1. To execute this API you need to have a Google Cloud Account 
2. Open the Extensions Tool in the dock on the left of your screen, and install the Google Cloud Auth extension from there."
3. Use the Extension to sign into your Google Cloud account.
4. Make sure the Google Cloud [Project has Vertex APIs enabled](https://console.cloud.google.com/vertex-ai)
5. **Set your [quota project](https://cloud.google.com/sdk/gcloud/reference/auth/application-default/set-quota-project) by running  "gcloud auth application-default set-quota-project <project-name>"**
6. You are now ready to execute the API. 

## Executing the API
1. Make any changes to the prompt as required. 
2. Run the code sample.
